## Projection Set Algorithms

- Proximal gradient methods are a powerful technique to solve non-convex inverse problems  
- Problem: find the intersection of two sets, only operating on one set a time

<summary> Higher-Dimensional Sets: Sudoku </summary>
  
  <ul>
    <li> Sudoku is a constraint satisfaction problem:</li>
      <ul>
        <li> Each row contains only one of each of the numbers 1-9 </li>
        <li> Each column contains only one of each of the numbers 1-9 </li>
        <li> Each 3x3 block contains only one of each of the numbers 1-9 </li>
        <li> Solution must be consistent with a (minimum) set of pre-defined clues </li>
      </ul>
    <li> Each constraint can be interpreted as a projection to a non-convex set </li>
    <li> This can be solved using iterative proximal gradient algorithms </li>
  </ul>
